<p><code>tslearn</code> is a Python package that provides machine learning tools for the analysis of time series.
This package builds on <code>scikit-learn</code>, <code>numpy</code> and <code>scipy</code> libraries.</p>
<p>If you would like to contribute to <code>tslearn</code>, please have a look at <a href="/rtavenar/tslearn/blob/master/CONTRIBUTING.md">our contribution guidelines</a>.</p>

<h1><a id="user-content-documentation-and-api-reference" class="anchor" aria-hidden="true" href="#documentation-and-api-reference"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Documentation and API reference</h1>
<p>The documentation, including a gallery of examples, is hosted at <a href="http://tslearn.readthedocs.io/en/latest/index.html" rel="nofollow">readthedocs</a>.</p>
<h1><a id="user-content-already-available" class="anchor" aria-hidden="true" href="#already-available"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Already available</h1>
<ul>
<li>A <code>generators</code> module provides Random Walks generators</li>
<li>A <code>datasets</code> module provides access to the famous UCR/UEA datasets through the <code>UCR_UEA_datasets</code> class</li>
<li>A <code>preprocessing</code> module provides standard time series scalers</li>
<li>A <code>metrics</code> module provides:
<ul>
<li>Dynamic Time Warping (DTW) (with Sakoe-Chiba band and Itakura parallelogram variants)</li>
<li>LB_Keogh</li>
<li>Global Alignment Kernel</li>
<li>Soft-DTW from Cuturi and Blondel</li>
</ul>
</li>
<li>A <code>neighbors</code> module includes nearest neighbor algorithms to be used with time series</li>
<li>An <code>svm</code> module includes Support Vector Machine algorithms with:
<ul>
<li>Standard kernels offered in <code>sklearn</code> (with adequate array reshaping done for you)</li>
<li>Global Alignment Kernel</li>
</ul>
</li>
<li>A <code>clustering</code> module includes the following time series clustering algorithms:
<ul>
<li>Standard Euclidean k-means (with adequate array reshaping done for you)
<ul>
<li>Based on <code>tslearn.barycenters</code></li>
</ul>
</li>
<li>DBA k-means from Petitjean <em>et al.</em>
<ul>
<li>Based on <code>tslearn.barycenters</code> that offers DBA facility that could be used for other applications than just
k-means</li>
</ul>
</li>
<li>Global Alignment kernel k-means</li>
<li>KShape clustering from Paparizzos and Gravano</li>
<li>Soft-DTW k-means from Cuturi and Blondel
<ul>
<li>Based on <code>tslearn.barycenters</code> that offers Soft-DTW barycenter computation</li>
</ul>
</li>
<li>It also provides a way to compute the silhouette coefficient for given clustering and metric</li>
</ul>
</li>
<li>A <code>shapelets</code> module includes an efficient implementation of the Learning Time-Series method from Grabocka <em>et al.</em>
<ul>
<li><strong>Warning:</strong> to use the <code>shapelets</code> module, an extra dependency is required: <code>keras</code></li>
</ul>
</li>
<li>A <code>piecewise</code> module includes standard time series transformations, as well as the corresponding distances:
<ul>
<li>Piecewise Aggregate Approximation (PAA)</li>
<li>Symbolic Aggregate approXimation (SAX)</li>
<li>1d-Symbolic Aggregate approXimation (1d-SAX)</li>
</ul>
</li>
</ul>
